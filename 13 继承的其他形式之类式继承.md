#继承的其他形式之类式继承

![](image/screenshot_1495466219598.png)

类：JS是没有类的概念的，在JS中可以把构造函数看做是类
```
function Aaa(){             // 父类
    this.name = '小明';
}

Aaa.prototype.showName = function(){
    alert(this.name);
};

function Bbb(){             // 子类

}

Bbb.prototype = new Aaa();  // 这一句话就是类式继承
var b1 = new Bbb();
b1.showName();              // 小明
```
