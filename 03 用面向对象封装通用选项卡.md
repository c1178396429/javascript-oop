#用面向对象封装通用选项卡

###改成面向对象原则：
1. 全局变量就是属性
2. 函数就是方法
3. onload中创建对象
4. 改this指向问题，在什么情况下this指向特别容易被修改（事件或者是定时器），尽量让面向对象中的this指向对象

// 复习this指向
```
oDiv.onclick = function(){
    this: oDiv      // 这里的this指的是oDiv
};

oDiv.onclick = show;
function show() {
    this: oDiv      // 这里的this指的还是oDiv
};

oDiv.onclick = function(){
    show();
};
function show() {
    this: window    // 这里的this指的就是window
}
```
